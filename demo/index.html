Multi Slider Component

This component provides you with an interactive vertical multiple slider, which helps on adjusting numeric values part of a whole. e.g. percentages.

Default behaviour

Initialised with 

buckets:[{key:'tory',value: 40}, {key: 'labour', value: 40}, {key: 'other', value: 20}]
And css rules 

.bucket.tory{
	background-color: #043140;
}
.bucket.labour{
	background-color: #F2EA61;
}
.bucket.other{
	background-color: #F9A532;
}
